33. La struttura delle medie#

source("../_common.R")

33.1. Utilizzo delle Medie nei Modelli SEM#

Nei modelli SEM, come nell’analisi fattoriale, ci focalizziamo principalmente sull’analisi delle covarianze tra le variabili. A differenza dell’analisi fattoriale tradizionale, i modelli SEM possono incorporare anche le medie delle variabili osservate e latenti. Questo aspetto è particolarmente utile in modelli SEM come i modelli CFA longitudinali, dove le ipotesi chiave riguardano le medie dei costrutti.

33.2. Interpretazione delle Intercette nei Modelli SEM#

In un modello SEM, l’intercetta di un indicatore (denotata con \( \tau \)) rappresenta la media stimata di quell’indicatore. \( \tau \) corrisponde al valore atteso dell’indicatore quando il fattore latente associato è zero. La formula generale per un indicatore \(y\) in un modello SEM è:

\[ y = \lambda \cdot \text{fattore latente} + \tau + \varepsilon, \]

dove:

  • \( y \) è il punteggio osservato dell’indicatore.

  • \( \lambda \) rappresenta il carico fattoriale.

  • \( \tau \) è l’intercetta, ovvero la media stimata dell’indicatore.

  • \( \varepsilon \) è l’errore di misura associato all’indicatore.

33.3. Struttura delle Medie nel Modello CFA#

In un modello CFA, la formula generale per la struttura delle medie è:

\[ \text{media(variabile latente)} = \Lambda \mu_{\text{lat}} + \tau, \]

dove:

  • \( \Lambda \) è la matrice dei carichi fattoriali.

  • \( \mu_{\text{lat}} \) è il vettore delle medie dei costrutti latenti.

  • \( \tau \) è il vettore delle intercette degli indicatori.

33.4. Struttura delle Medie e lavaan#

Per stimare le intercette in un modello SEM con lavaan, si richiedono i dati originali o una matrice di covarianza, oltre alle medie di tutte le variabili. La sintassi meanstructure = TRUE in lavaan serve a incorporare automaticamente la costante “1” in tutte le equazioni del modello, facilitando il calcolo delle intercette per le variabili endogene.

    visual =~ NA*x1 + x2 + x3
    textual =~ NA*x4 + x5 + x6
    speed =~ NA*x7 + x8 + x9

    visual ~~ 1*visual

    speed ~~ 1 * speed

fit <- cfa(hs_model,
    data = HolzingerSwineford1939,
    meanstructure = TRUE

33.5.1. Interpretazione delle Medie Stimate#

La media dei punteggi osservati per gli indicatori (x1, x2, x3, ecc.) viene calcolata attraverso le intercette stimate dal modello. È fondamentale distinguere tra la media empirica, calcolata direttamente dai dati, e la media predetta dal modello. La media predetta degli indicatori in un modello dove la media dei costrutti latenti è fissata a zero è influenzata esclusivamente dalle loro intercette.

33.5.2. Calcolo delle Medie Osservate e Predette in R#

Consideriamo gli indicatori x1, x2, x3. Per calcolare la media osservata di questi indicatori, usiamo le loro intercette stimate.

intercepts <- params$est[params$op == "~1"][1:9] # Intercette degli indicatori (τ)

Questo ci fornisce le intercette degli indicatori:

intercepts |> print()

[1] 4.935770 6.088040 2.250415 3.060908 4.340532 2.185572 4.185902 5.527076
[9] 5.374123

Per ottenere la media osservata dei punteggi di x1, x2, x3, calcoliamo la media aritmetica delle loro intercette:

 mean_observed_scores <- mean(intercepts[1:3])
 print(mean_observed_scores)

[1] 4.424742

Questo valore rappresenta la media osservata calcolata come la media aritmetica delle intercette di x1, x2, x3. Nel contesto del nostro modello SEM, dove la media dei costrutti latenti è fissata a zero, la media predetta degli indicatori corrisponderà alla media osservata:

mean((HolzingerSwineford1939$x1 + HolzingerSwineford1939$x2 + HolzingerSwineford1939$x3) / 3)
4.42474160196013
mean_predicted_scores <- mean_observed_scores
print(mean_predicted_scores)

[1] 4.424742

33.5.3. Medie di Costrutti Latenti Non Zero#

In situazioni in cui le medie dei costrutti latenti non sono fissate a zero, la media predetta degli indicatori sarebbe influenzata sia dalle intercette sia dai carichi fattoriali. Per esempio, se la media del costrutto latente fosse diversa da zero, l’equazione per calcolare la media di un indicatore (come x1) includerebbe il contributo del costrutto latente:

\[ \text{media predetta}(x1) = \mu_{\text{latente}} \cdot \lambda_{x1} + \tau_{x1}, \]

dove:

  • \(\mu_{\text{latente}}\) è la media stimata del costrutto latente.

  • \(\lambda_{x1}\) è il carico dell’indicatore x1.

  • \(\tau_{x1}\) è l’intercetta stimata dell’indicatore x1.

Esaminiamo un esempio nel quale le medie dei fattori latenti non sono fissate a zero. Per ottenere questo risultato è necessario identificare il modello introducendo due vincoli:

  • l’intercetta degli indicatori è fissata a zero;

  • una delle intercette delle variabili latenti è fissata a zero.

hs_model <- "
    visual =~ NA*x1 + x2 + x3
    textual =~ NA*x4 + x5 + x6
    speed =~ NA*x7 + x8 + x9

    visual ~~ 1*visual
    textual ~~ 1*textual
    speed ~~ 1*speed

    x1 ~ 0*1 # Setting the intercepts of the manifest
    x4 ~ 0*1 # variables to zero
    x7 ~ 0*1

    visual ~ 0*1 # Setting the mean of visual to zero
    textual ~ 1 # freely estimating the mean of textual
    speed ~ 1 # freely estimating the mean of speed
"

# Fit del modello con la struttura delle medie
fit <- cfa(hs_model, data = HolzingerSwineford1939, meanstructure = TRUE)

params <- parameterEstimates(fit)
print(params)
